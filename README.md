# 👁️ VisionForge-Deep-CNN-Classifier

## 📌 Overview
This project implements a **Convolutional Neural Network (CNN)** for image classification on the **CIFAR-10 dataset** using Deep Learning.

The model learns to classify images into multiple categories by extracting hierarchical features, showcasing the effectiveness of CNNs in computer vision tasks.

---

## ⚙️ Model Architecture

### 🔹 Convolutional Neural Network (CNN)
• Convolutional layers for feature extraction  
• ReLU activation for non-linearity  
• MaxPooling layers for dimensionality reduction  
• Fully connected layers for classification  
• Softmax output layer for multi-class prediction  

---

## ✨ Features
• CIFAR-10 dataset for multi-class image classification  
• Data normalization and preprocessing  
• CNN-based deep learning model  
• Training and validation pipeline  
• Loss and accuracy tracking  
• Model evaluation on test data

## 🎮 Workflow
